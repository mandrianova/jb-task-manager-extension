task-cli.sh lets Claude manage tasks from the terminal.

Quick Start

Three steps, three minutes

1

Build & install

Run ./gradlew buildPlugin, then Settings → Plugins → Install from Disk.

2

Install Claude skills

Click Install Claude Skills in the toolbar — or copy to ~/.claude/skills/ globally.

3

Create & execute

Press + to create with /task-create, then ▶ to run with /task-execute.

Ready to try?

Clone the repo, build, and start shipping tasks with Claude.

$ ./gradlew buildPlugin
